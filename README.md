## サイト名
「 （仮）絵本の世界へようこそ 」

## サイト概要
　絵本に関するサービス。（絵本の紹介・購入・レビューなどの機能）
 
## 機能
※作成を予定している機能（もっと機能を充実させていく）

* ユーザー、管理者用の認証システム（ログイン・ログアウト画面）
* 新着絵本・特集絵本・年齢別の絵本を紹介する
* 気に入った絵本があれば、購入できるシステム
